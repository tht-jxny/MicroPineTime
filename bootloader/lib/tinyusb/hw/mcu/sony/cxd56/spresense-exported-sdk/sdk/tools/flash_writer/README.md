# Flash Writer

This tool can easy to flash spk files into spresense board by using X-Modem.

## Requires

1. Python 3.x
1. pySerial
1. pyinstaller

## Tools installation

$ pip install pyserial
$ pip install pyinstaller

## How to create exection file

$ pyinstaller --onefile scripts/flash_writer.py
